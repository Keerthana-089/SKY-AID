# SkyAid – Disaster Relief Resource Platform

## Overview

SkyAid is a web-based disaster relief platform designed to help manage emergency resource requests during natural disasters.
The platform allows users to request essential supplies such as food, water, medicines, and first aid, while enabling administrators to monitor and manage resource distribution.

The goal of SkyAid is to improve coordination during disaster situations and ensure faster access to critical resources.

---

## Features

* User login and authentication interface
* Dashboard for viewing resource requests
* Emergency resource categories such as food, water, medicines, and first aid
* Request submission system for disaster victims
* Organized UI pages for navigation between different services
